Title: Chapter 163 - အစွမ်းကုန် စွန့်စားခြင်း!

"ဘာဆာကာတွေက ငါတို့ ထာဝရရှင်သန်သူတွေရဲ့ မန္တရား သင်္ကေတတွေကို ဘယ်လိုလုပ်ပြီး နားလည်နိုင်မှာလဲ? ဒါက လွန်ခဲ့တဲ့ အချိန်ကတည်းက အတိုင်းအတာ လေးခုစလုံး ပြီးပြည့်စုံတဲ့ တံဆိပ်ခတ်နှိပ်မှု မဟုတ်တော့ဘူး၊ ၎င်းရဲ့ အလယ်ဗဟိုမှာ နောက်ထပ် အတိုင်းအတာဝက်တစ်ခု ကျန်ရှိနေသေးတာ ဖြစ်တယ်!"

စုမင်၏ အမြင်အာရုံများ ဝေဝါးသွားခဲ့သည်။ ၎င်းတို့ ပြန်လည် ရှင်းလင်းသွားချိန်၌ သူသည် မှောင်မိုက်လှသော နေရာတစ်ခုတွင် ပေါ်ထွက်လာခဲ့သည်။ ၎င်းမှာ တောင်ဂူတစ်ခု ဖြစ်သည်။ သူ၏ ဘေးပတ်လည်၌ မည်သည့်အလင်းရောင်မျှ ရှိမနေခဲ့ပေ။ သူသည် သေးငယ်လှသော အမြင်အာရုံကိုသာ အသုံးပြုနိုင်ခဲ့သည်။

"မြန်မြန်... ငါ့ကို ပြန်ပို့ပေးစမ်း... ငါ လွန်ခဲ့တဲ့ အချိန်တုန်းက ပြင်းထန်စွာ ဒဏ်ရာရခဲ့ပြီး ပြန်မကောင်းနိုင်ခဲ့ဘူး... ငါ ငါ့ရဲ့ အစစ်အမှန် ခန္ဓာကိုယ်ကို စတေးခဲ့ပြီး ယခု ဤပုံတူခန္ဓာကိုယ်တစ်ခုတည်းသာ ကျန်ရှိတော့တာ ဖြစ်တယ်... ငါ သိပ်ပြီး အကြာကြီး တောင့်ခံနိုင်မှာ မဟုတ်ဘူး... ငါ့ရဲ့ အစစ်အမှန် ခန္ဓာကိုယ် သေဆုံးသွားတာနဲ့ ငါ ဤနေရာကနေ မထွက်ခွာနိုင်ခဲ့ရင် ငါလည်း သေဆုံးရလိမ့်မယ်!"

"ငါ့ရဲ့ ပုံတူခန္ဓာကိုယ်ကသာ ဘာဆာကာတွေရဲ့ နယ်မြေကနေ ထွက်ခွာသွားနိုင်ပြီး ငါတို့ ထာဝရရှင်သန်သူတွေရဲ့ မတူညီတဲ့ နိယာမ Law တွေကို သုံးပြီး ငါ့ရဲ့ အစစ်အမှန်ခန္ဓာကိုယ်နဲ့ ပတ်သက်မှုတွေကို ဖြတ်တောက်နိုင်မှသာ ငါ ဆက်လက် အသက်ရှင်နိုင်မှာ ဖြစ်တယ်"

စုမင်၏ ဘေးနားမှ ပြင်းထန်လှသော အသက်ရှူသံများ ထွက်ပေါ်လာခဲ့ပြီး ဟန်ကုန်းသည် သူ၏ ပခုံးကို တင်းကျပ်စွာ ဆုပ်ကိုင်ထားခဲ့သည်။ သူ၏ မျက်နှာမှာ စုမင်၏ မျက်နှာနှင့် ခုနစ်လက်မခန့်သာ ကွာဝေးလှပေသည်။ သူသည် စုမင်၏ မျက်ဝန်းများကို ရူးသွပ်စွာဖြင့်လည်းကောင်း၊ စိုးရိမ်ပူပန်စွာဖြင့်လည်းကောင်း စိုက်ကြည့်နေခဲ့သည်။ ထိုအခိုက်အတန့်၌ သူသည် စိတ်ပျက်အားငယ်လုနီးပါး ဖြစ်နေခဲ့ပြီဖြစ်ပြီး ယခင်က ရှိခဲ့ဖူးသော တည်ငြိမ်မှုများ လုံးဝ မရှိတော့ပေ။

စုမင် ခေတ္တမျှ တိတ်ဆိတ်သွားပြီးနောက် ဖြည်းညှင်းစွာ ပြောဆိုလိုက်သည်။ "ငါ့မှာ မင်းကို ပြန်ပို့ပေးနိုင်မယ့် နည်းလမ်း လုံးဝ မရှိဘူး"

"ငါ နောက်တစ်ကြိမ် ထပ်ပြောမယ်... ငါ့ကို ပြန်ပို့ပေးစမ်း!!" ဟန်ကုန်း ကျယ်လောင်စွာ ဟောက်လိုက်ပြီး စုမင်၏ ပခုံးနှစ်ဖက်ကို လှမ်းဆွဲလိုက်ကာ သူ၏ မျက်နှာပေါ်တွင် သတ်ဖြတ်လိုသောအငွေ့အသက်များ ပေါ်လာခဲ့သည်။

"အကယ်၍ မင်း ငါ့ကို မယုံရင် ငါ့ရဲ့ ခန္ဓာကိုယ်ကို လုယူပြီး ငါ့ရဲ့ အမှတ်ရစရာ Memory တွေကို ရှာဖွေကြည့်ရှုနိုင်ပါတယ်၊ ငါ မင်းကို လိမ်ညာနေသလားဆိုတာကိုပေါ့..."

စုမင်၏ မျက်နှာအမူအရာမှာ အလွန် တည်ငြိမ်နေခဲ့ပြီး ဟန်ကုန်းအား စိုက်ကြည့်လိုက်သည်။ ဤသည်မှာ သူ၏ ရည်ရွယ်ချက် ဖြစ်သည်!

သူသည် လှိုဏ်ခေါင်း၏ အဆုံး၌ ရှိနေခဲ့စဉ်ကများစွာသော အရာများကို သိရှိနားလည်ခဲ့သည်။ သူသည် မိမိကိုယ်တိုင် အမှတ်ရစရာအချို့ကို တကယ်ပဲ မေ့လျော့နေသလားဟူသည်ကို မသေချာသဖြင့် ၎င်းကို လျစ်လျူရှု၍ မရနိုင်ခဲ့ပေ။ သူသည် ဤအရာအားလုံးကို ကောင်းစွာ နားလည်လိုပေသည်။

သို့သော်လည်း စုမင်သည် မိမိ၏ လက်ရှိအစွမ်းဖြင့် မိမိအပေါ် ဘာတွေ ဖြစ်ပျက်ခဲ့သနည်းဟူသည်ကို သိရှိရန် လုံးဝ မဖြစ်နိုင်ကြောင်း ကောင်းစွာ နားလည်ထားသည်။ ထို့ကြောင့်လည်း သူ ဟန်တောင်တန်း ဘိုးဘေးကို သတိရမိလိုက်ပြီး ဟော်ဖုန်း ပြောပြခဲ့ဖူးသော... ခန္ဓာကိုယ်လုယူခြင်း (Possession) အကြောင်းကို စဉ်းစားမိလိုက်ခြင်း ဖြစ်သည်။

ခန္ဓာကိုယ်လုယူခြင်း ဖြစ်စဉ်အတွင်း အရေးကြီးသော အချက်တစ်ခု ရှိနေခဲ့သည် - ၎င်းမှာ လုယူနေစဉ်အတွင်း အခြားသူ၏ အမှတ်ရစရာ Memory များကို အတင်းအကြပ် ရှာဖွေကြည့်ရှုနိုင်ခြင်း ဖြစ်သည်!

ဟော်ဖုန်းသည် ၎င်းကို မလုပ်ဆောင်နိုင်သော်လည်း စုမင်သည် ဟန်တောင်တန်း ဘိုးဘေးအနေဖြင့် ၎င်းကို သေချာပေါက် လုပ်ဆောင်နိုင်လိမ့်မည်ဟု ယုံကြည်ထားခဲ့သည်။

‘ငါ ဘာဖြစ်လို့ ဟန်ချန်ဇီရဲ့ မျက်ဝန်းတွေထဲမှာ သနားစနိုးဖြစ်နေတဲ့အကြည့်ကို မြင်ခဲ့ရသလဲဆိုတာ သိချင်တယ်... ငါ ဘာဖြစ်လို့ နန်ထျန်းက ဘာဆာကာနတ်ဘုရားရဲ့ လက်ထဲမှာ မီးတောက်ဘာဆာကာတွေ မျိုးဖြုတ်သတ်ဖြတ်ခံခဲ့ရတဲ့အကြောင်းကို မပြောခဲ့ရသလဲဆိုတာ သိချင်တယ်... ငါ နီရဲတဲ့ လမ်းပေါ်မှာ သွားတုန်းက ကြားခဲ့ရတဲ့ စကားလုံးတွေနဲ့ အေးစက်လှတဲ့ အကြည့်တွေကြောင့် ဘာဖြစ်လို့ စိုးရိမ်ထိတ်လန့်ပြီး ကြောက်ရွံ့ခဲ့ရသလဲဆိုတာကို သိချင်တယ်...’

‘ငါ တကယ်ပဲ အမှတ်ရစရာ Memory အချို့ကို မေ့လျော့နေတာလား၊ အကယ်၍ ဟုတ်ခဲ့ရင် ၎င်းက ဘယ်အချိန်တုန်းက ဖြစ်ပျက်ခဲ့တာလဲဆိုတာ သိချင်တယ်...’

‘ငါ ပျောက်ဆုံးနေတဲ့ အမှတ်ရစရာ Memory တွေထဲမှာ ဘာတွေ ဖြစ်ပျက်ခဲ့သလဲဆိုတာ သိချင်တယ်...’

‘ငါ့ရဲ့ အကြီးအကဲနဲ့ တောင်တန်းမည်းအကြောင်း Memory တွေအားလုံးက တကယ်ပဲ အိပ်မက်တစ်ခုလားဆိုတာ သိချင်တယ်...’

စုမင် မျက်လုံးများကို ပိတ်လိုက်ပြီး ၎င်းတို့ကို ပြန်လည် ဖွင့်လိုက်သည်။ သူသည် ပြင်းထန်စွာ အသက်ရှူနေသော ဟန်ကုန်းအား စိုက်ကြည့်လိုက်တော့သည်။

"ငါ့ကို ကူညီပါ... ငါ့ရဲ့ အမှတ်ရစရာ Memory တွေထဲမှာ ငါ ဘာတွေ ဆုံးရှုံးခဲ့ရသလဲဆိုတာကို ငါ့ကို ပြောပြပေးပါ၊ ဒါဆို မင်းလည်း ငါ မင်းကို မင်းရဲ့ အိမ်ကို ဘာဖြစ်လို့ ပြန်မပို့ပေးနိုင်ရသလဲဆိုတဲ့ အကြောင်းရင်းကို ကိုယ်တိုင် မြင်တွေ့ရလိမ့်မယ်... ငါ မင်းကို မလိမ်ညာဘူးဆိုတာကိုလည်း မင်း သိရလိမ့်မယ်"

"ငါ့ကို ကူညီပါ... ငါ့ကို ပြောပြပါ... ငါ ဘယ်သူလဲဆိုတာကိုပေါ့..." စုမင် တိုးညှင်းစွာ ရေရွတ်လိုက်ပြီး သူ၏ မျက်ဝန်းများထဲတွင် ပြတ်သားသောအကြည့်တစ်ခု ပေါ်လာခဲ့တော့သည်။

ဟန်ကုန်း သူ့အား စိုက်ကြည့်လိုက်သည်။ သူ မည်သည့်အကြောင်းကြောင့်မသိ စုမင်၏ အကြည့်အောက်တွင် သူ၏ နှလုံးသားအတွင်း၌ ဖော်ပြရန် ခက်ခဲလှသော ကြောက်ရွံ့မှုတစ်ခု ပေါ်ထွက်လာခဲ့သည်။ သူသည် စုမင်၏ တည်ငြိမ်မှုကိုများစွာ ကြောက်ရွံ့နေခဲ့ခြင်း ဖြစ်သည်။ သူသည် မိမိ၏ ဘဝတွင် မိမိ၏ ခန္ဓာကိုယ်ကို လာရောက် လုယူရန် ကိုယ်တိုင် တောင်းဆိုသောလူမျိုးကို တစ်ခါမှ မကြုံတွေ့ဖူးသဖြင့် ကြောက်ရွံ့နေခဲ့ခြင်း ဖြစ်သည်။

"ရိုးရိုးရှင်းရှင်း ခန္ဓာကိုယ်လုယူခြင်းနဲ့ ဝိညာဉ်ရှာဖွေခြင်း (Soulsearch) က ငါတို့ကို အမှတ်ရစရာ Memory အပြည့်အစုံကို မမြင်တွေ့စေနိုင်ဘူး၊ အပိုင်းအစတွေကိုပဲ မြင်တွေ့ရလိမ့်မယ်... အကယ်၍ မင်း ငါ့ကို ပြန်မပို့ပေးဘဲ မင်းရဲ့ ခန္ဓာကိုယ်ကို လုယူခွင့်ပေးမယ်ဆိုရင်တော့ ငါ မင်းကို ကူညီပါ့မယ်!"

"ငါ့ရဲ့ ဤပုံတူခန္ဓာကိုယ်ကို ဘာဆာကာကျင့်ကြံသူတစ်ယောက်ရဲ့ ခန္ဓာကိုယ်နဲ့ သန့်စင်ထားခဲ့တာ ဖြစ်တယ်... ထိုလူက အသက်ရှင်စဉ်တုန်းက အရိုးစတေးခြင်းအဆင့်ရဲ့ စွမ်းအားကို ရရှိထားခဲ့ပြီးပြီ... အကယ်၍ မင်း တကယ်ပဲ ငါ့ကို ကူညီစေချင်ရင် ငါ မင်းရဲ့ ခန္ဓာကိုယ်ကို ငါ့ရဲ့ ပုံတူခန္ဓာကိုယ်အဖြစ် ပြန်လည် သန့်စင်ရလိမ့်မယ်!"

"အကယ်၍ ငါ သေဆုံးရင် မင်းလည်း ငါနဲ့အတူ သေဆုံးရလိမ့်မယ်... အကယ်၍ မင်း မသေဆုံးခဲ့ရင်တော့ ဒါက မင်းအတွက် ကောင်းမွန်တဲ့ အခွင့်အရေး ဖြစ်လာလိမ့်မယ်... ငါ မင်းရဲ့ ဆန္ဒကို ဖြည့်ဆည်းပေးမယ်!"

ဟန်ကုန်းသည် နှစ်ပေါင်းများစွာ အသက်ရှင်ခဲ့သူဖြစ်သဖြင့် စုမင်၏ အစီအစဉ်ကို မရိပ်မိဘဲ မနေခဲ့ပေ၊ သို့သော်လည်း သူ၏ထံတွင် အချိန်လုံးဝ မရှိတော့ပေ... သူ၏ မျက်ဝန်းများထဲတွင် ရက်စက်ယုတ်မာသောအကြည့်တစ်ခု ပေါ်လာခဲ့သည်။ သူသည် နှစ်ပေါင်း ၈၀၀၀ ကြီးများတောင် စောင့်ဆိုင်းခဲ့ရသော်လည်း နောက်ဆုံးရလဒ်က ဤသို့ ဖြစ်လာခဲ့သည်။ သူ အိမ်မပြန်နိုင်တော့သလို သူ၏ မွေးရပ်မြေသို့လည်း မပြန်နိုင်တော့ပေ။

သူသည် အချိန်မရွေး သေဆုံးသွားရနိုင်ပေသည်။ သူ၏ ပြင်ပကမ္ဘာရှိ အစစ်အမှန်ခန္ဓာကိုယ် သေဆုံးသွားသည်နှင့် သူလည်း သေဆုံးရလိမ့်မည် ဖြစ်သည်။ ဤသို့ သေဆုံးသွားမည့်အစား သူသည် သူ့အား မျှော်လင့်ချက်များပေးကာ တစ်ချိန်တည်းမှာပင် ခြေမှုန်းဖျက်ဆီးပစ်လိုက်သော ဤကံကြမ္မာ (Destiny) ၏ ခန္ဓာကိုယ်ကို ပြန်လည် သန့်စင်ပစ်လိုက်ခြင်းကသာ သာ၍ ကောင်းမွန်ပေလိမ့်မည်။ သူသည် သူတို့နှစ်ဦးလုံး တူတူ သေဆုံးသွားစေရန် ပြုလုပ်လိမ့်မည် ဖြစ်သည်။ သို့သော်လည်း မည်သို့ပင်ဖြစ်စေ သူသည် စုမင်၏ အမှတ်ရစရာ Memory အားလုံးကို မြင်တွေ့ရလိမ့်မည် ဖြစ်သည်။

သူသည် စုမင် သဘောတူသည်ကိုပင် စောင့်ဆိုင်းမနေခဲ့ပေ။ ဟန်ကုန်းသည် သူ၏ ညာလက်ကို အလျင်အမြန် မြှောက်ကာ သူ၏ ဝမ်းဗိုက်ကို ရိုက်ချလိုက်တော့သည်။ သူ၏ ခန္ဓာကိုယ် ပြင်းထန်စွာ တုန်ယင်သွားခဲ့ပြီး ကျယ်လောင်သော ပေါက်ကွဲသံနှင့်အတူ စုမင်၏ မျက်စိရှေ့တွင် နီရဲသော သွေးမြူထုကြီးတစ်ခုအဖြစ် ပြောင်းလဲသွားခဲ့တော့သည်။

ထိုသွေးမြူထုသည် စုမင်၏ ခန္ဓာကိုယ်ကို လွှမ်းခြုံသွားခဲ့ပြီး ရွှေရောင်အလင်းတန်းတစ်ခုကို ထုတ်လွှတ်လိုက်သည်။ လက်ဖဝါးခန့်ရှိသော သေးငယ်လှသည့် ရွှေရောင်လူသားတစ်ဦးသည် ထိုမြူထုအတွင်းမှ ပြေးထွက်လာခဲ့ပြီး စုမင်၏ မျက်ခုံးနှစ်ခုကြားရှိ နေရာဆီသို့ တဟုန်ထိုး ပြေးဝင်သွားခဲ့တော့သည်။ ထိုလူသားမှာ ဟန်ကုန်း ဖြစ်သော်လည်း သူ၏ စွမ်းအားမှာ ဟော်ဖုန်းနှင့် နှိုင်းယှဉ်၍မရနိုင်လောက်အောင် မြင့်မားလှပေသည်။

ထိုသေးငယ်သော လူသားသည် ဝိညာဉ်ခန္ဓာ မဟုတ်တော့ဘဲ လုံးဝ ရုပ်လုံးပေါ်နေပြီဖြစ်သည့် မူလဝိညာဉ်ကလေးငယ် (Spirit Infant) ထက်များစွာ အစွမ်းထက်လှသော မူလနတ်ဘုဉ်းဝိညာဉ် (Nascent Divinity) ဖြစ်နေခဲ့ပြီ ဖြစ်သည်။ သူ၏ ခန္ဓာကိုယ်အတွင်း၌ ရွှေရောင် ကျောရိုးတစ်ခုကိုပင် မြင်တွေ့နိုင်ပေသည်။

ထိုကျောရိုးမှာ မှေးမှိန်နေသော်လည်း ၎င်းထံမှ ကြမ်းတမ်းပြီး ရိုင်းစိုင်းလှသော အငွေ့အသက်တစ်ခု ထွက်ပေါ်နေခဲ့သည်။ ၎င်းမှာ အရိုးစတေးခြင်းအဆင့်ရှိ အစွမ်းထက် ဘာဆာကာကျင့်ကြံသူတစ်ဦး၏ ခန္ဓာကိုယ်အတွင်းမှ ပထမဆုံး ပြန်လည် ဆန်းသစ်လာခဲ့သော အရိုး ဖြစ်သည်!

ဤသည်မှာ စုမင် ခန့်မှန်းထားခဲ့သည်နှင့် အနည်းငယ် ကွဲပြားခြားနားသော်လည်း နောက်ဆုံးရလဒ်က တူညီသဖြင့် သူ ဆက်လက် မငြင်းဆန်ခဲ့ပေ။ သူသည် ဟန်ကုန်း၏ မူလနတ်ဘုဉ်းဝိညာဉ် သူ့ဆီသို့ ဆင်းသက်လာသည်ကို ကြည့်ရှုရင်း မျက်လုံးများကို ပိတ်လိုက်တော့သည်။

‘လောကမှာ အရာအားလုံးကို တွက်ချက်ထားနိုင်တဲ့ ပြီးပြည့်စုံတဲ့ အစီအစဉ်ဆိုတာ မရှိပါဘူး... အမှားအယွင်းတွေ ရှိမှသာ အမှန်တရားကို ရှာဖွေတွေ့ရှိနိုင်မယ့် အခွင့်အရေးလည်း ရှိတာ ဖြစ်တယ်... ငါ စွန့်စားရမယ်!’

သူ မျက်လုံးများကို ပိတ်လိုက်သည့် အခိုက်အတန့်၌ ဟန်ကုန်း၏ မူလနတ်ဘုဉ်းဝိညာဉ်သည် သူ၏ မျက်ခုံးနှစ်ခုကြားသို့ ထိတွေ့ကာ အတွင်းသို့ လျင်မြန်စွာ ဝင်ရောက်ပျောက်ကွယ်သွားခဲ့တော့သည်။ ၎င်း ဝင်ရောက်ပျောက်ကွယ်သွားခဲ့သည့် အခိုက်အတန့်၌ စုမင်၏ ခန္ဓာကိုယ် ပြင်းထန်စွာ တုန်ယင်သွားခဲ့တော့သည်။ မျက်နှာဖုံးအောက်ရှိ သူ၏ မျက်နှာပေါ်တွင် နာကျင်လှသောအမူအရာများ ပေါ်လာခဲ့သည်။

ဤနာကျင်မှုသည် ဟော်ဖုန်း သူ့အား ခန္ဓာကိုယ် လာရောက်လုယူစဉ်ကထက် အဆပေါင်းများစွာ သာ၍ ပြင်းထန်လှပေသည်။ ထိုနာကျင်မှု နှစ်ခုကို နှိုင်းယှဉ်၍ပင် မရနိုင်ဘဲ ကောင်းကင်နှင့် မြေပြင်ကဲ့သို့ ကွာခြားလှပေသည်။ သူ၏ ခန္ဓာကိုယ်တစ်ခုလုံးတွင် ရေတွက်၍မရနိုင်လောက်အောင်များပြားလှသော အပ်များဖြင့် အပေါက်တိုင်းကို ထိုးနှက်ခံနေရသကဲ့သို့ နာကျင်နေရုံသာမက သူ၏ စိတ်ထဲ၌လည်း ဧရာမ တောင်စဉ်တောင်တန်းကြီးတစ်ခုက ၎င်းကို ဖိနှိပ်ထားသကဲ့သို့ ခံစားရစေပြီး သူ့အား အစိတ်စိတ်အမြွှာမြွှာ ကွဲကြေပျက်စီးသွားတော့မည့်အလား ခံစားရစေသည်။

ထိုပြင်းထန်လှသော နာကျင်မှုအောက်တွင် မျက်နှာဖုံးအောက်ရှိ သူ၏ မျက်နှာပေါ်မှ သွေးစက်များ ချက်ချင်း စိမ့်ထွက်လာခဲ့ကြသည်။ ထိုသွေးများသည် သူ၏ နှာခေါင်း၊ မျက်လုံး၊ နားနှင့် ပါးစပ်တို့မှ စီးကျလာခဲ့ကြခြင်း ဖြစ်သည်။

ထိုပြင်းထန်လှသော နာကျင်မှုကို ဖော်ပြရန် ခက်ခဲလှသော်လည်း စုမင်သည် မည်သည့်အသံမျှ မထွက်ခဲ့သလို ရူးသွပ်သွားခြင်းလည်း မရှိခဲ့ပေ။ သူသည် ရေတွင်းထဲရှိ ငြိမ်သက်နေသော ရေပြင်ကြီးကဲ့သို့ပင် အလွန် တည်ငြိမ်လှသော မျက်ဝန်းများဖြင့် ထိုနေရာ၌ ဆက်လက် ထိုင်နေခဲ့သည်။

သူသည် ထိုနာကျင်မှုကို တိတ်ဆိတ်စွာ တောင့်ခံထားခဲ့ပြီး ဟန်ကုန်း၏ ခန္ဓာကိုယ်လုယူခြင်း သို့မဟုတ် သူ၏ ခန္ဓာကိုယ်ကို ပုံတူခန္ဓာကိုယ်အဖြစ် ပြန်လည် သန့်စင်နေမှုကို တိတ်တဆိတ် ခံယူနေခဲ့သည်။ သူသည် အရာအားလုံးကို တိတ်ဆိတ်စွာသာ တောင့်ခံနေခဲ့သည်။

သူ၏ အမြင်အာရုံတစ်ခုလုံး ဝေဝါးသွားခဲ့ပြီး သူသည် ယခင်က ရောက်ရှိခဲ့ဖူးသော နေရာသို့ တစ်ဖန် ပြန်လည် ရောက်ရှိလာခဲ့ပြန်သည် - မြူနှင်းများ ဖုံးလွှမ်းနေသော ကောင်းကင်ယံကြီးနှင့် ပေရာချီ ကျယ်ဝန်းလှသော ဗလာကျင်းနေသည့် နေရာလွတ် ဖြစ်သည်။

စုမင်သည် ထိုအားနည်းလှသော ဝိညာဉ်လုံးအသွင်ဖြင့် ရှိနေဆဲ ဖြစ်သည်။ သူသည် လေထုထဲ၌ မျောလွင့်နေစဉ် သူ၏ ရင်းနှီးလှသော ခံစားချက်နှင့် သူ၏ ပြတ်သားမှုတို့ကြောင့် သူ တုန်ယင်သွားခြင်း သို့မဟုတ် ကြောက်ရွံ့စုန်းစုန်းမြုပ်သွားခြင်း မရှိခဲ့ပေ။ သူသည် သူ၏ အကြည့်ကို ဟန်ကုန်း၏ မူလနတ်ဘုဉ်းဝိညာဉ်ဆီသို့ ပို့လွှတ်လိုက်ရာ ၎င်းထံမှ စူးရှလှသော ရွှေရောင်အလင်းတန်းများ ထွက်ပေါ်နေခဲ့သဖြင့် သူ့အား အရည်ပျော်သွားစေတော့မည့်အလား ရှိနေခဲ့သည်။

ထိုမူလနတ်ဘုဉ်းဝိညာဉ်သည် ပေပေါင်းများစွာ မြင့်မားလှပေသည်။ စုမင်နှင့် နှိုင်းယှဉ်ပါက သူသည် ကောင်းကင်နှင့် မြေပြင်ကိုပင် ဖုံးလွှမ်းထားနိုင်သော ဧရာမ လူသားကြီးတစ်ဦးကဲ့သို့ ဖြစ်နေခဲ့သည်။ သူ ဆင်းသက်လာချိန်၌ ပေရာချီ ကျယ်ဝန်းလှသော ထိုနေရာလွတ်သည်လည်း ဤပြင်းထန်လှသော ဖိအားကို မတောင့်ခံနိုင်တော့ဘဲ ပြိုကျပျက်စီးတော့မည့်အလား ပြင်းထန်စွာ လှုပ်ခါသွားခဲ့တော့သည်။

ထိုစူးရှလှသော ရွှေရောင်အလင်းတန်းနှင့် ဧရာမ လူသားကြီးသည် စုမင်အား မုန်တိုင်းအတွင်း ပိတ်မိနေသကဲ့သို့ ခံစားရစေပြီး ထိုလူသားကြီးသည် သူနှင့် မနီးကပ်သေးခင်မှာပင် သူ ချက်ချင်း ကွဲကြေပျက်စီးသွားတော့မည့်အလား ခံစားရစေသည်။

‘ဒီစိတ်နယ်ပယ်က ဘာဖြစ်လို့ ဒီလိုမျိုး ဖြစ်နေရတာလဲ..? ဘာဖြစ်လို့ ဒီလောက်အထိ မြူနှင်းတွေ အများကြီး ရှိနေရတာလဲ?!’

ဟန်ကုန်း သူ၏ ပတ်ဝန်းကျင်ကို ရှင်းလင်းစွာ မြင်တွေ့လိုက်ရချိန်၌ သူ၏ မူလနတ်ဘုဉ်းဝိညာဉ်၏ မျက်နှာပေါ်တွင် တုန်လှုပ်ခြောက်ခြားစရာကောင်းလှသော အမူအရာတစ်ခု ပေါ်လာခဲ့သည်။ သူသည် ၎င်း၏ သဘောတရားကို နားလည်သွားပုံရပြီး စုမင်အား ရှုပ်ထွေးလှသော မျက်ဝန်းများဖြင့် လျင်မြန်စွာ လှည့်ကြည့်လိုက်တော့သည်။

"ငါ မင်းရဲ့ တောင်းဆိုချက်ကို ဖြည့်ဆည်းပေးမယ်!"

ဟန်ကုန်း၏ ပါးစပ်မှ ထိုစကားလုံးများ ထွက်ပေါ်လာခဲ့သည့် အခိုက်အတန့်၌ ထိုဧရာမ မူလနတ်ဘုဉ်းဝိညာဉ်သည် သူ၏ ပါးစပ်ကို အစွမ်းကုန် ဟလိုက်ပြီး စုမင် ရှိရာအရပ်ဆီသို့ ပြင်းထန်စွာ ဆွဲယူလိုက်တော့သည်။ ချက်ချင်းပင် စုမင်၏ ဝိညာဉ်လုံးသည် ဟန်ကုန်းဆီသို့ တဟုန်ထိုး ပျံသန်းသွားခဲ့တော့သည်။

သူ ခန္ဓာကိုယ်လုယူခြင်း ခံရတော့မည်ဖြစ်သော်လည်း ဤသည်မှာ စုမင်၏ ကိုယ်ပိုင် စိတ်နယ်ပယ် ဖြစ်နေဆဲ ဖြစ်သည်။ သူသည် သူ၏ရှေ့ရှိ မြူနှင်းများ အလျင်အမြန် ပါးလွှာသွားသည်ကို မြင်တွေ့လိုက်ရပြီး သူ တောင်ဘက်နံနက်ခင်းဒေသ၌ ရှိနေခဲ့စဉ်က အမှတ်ရစရာ Memory များသည် လျင်မြန်စွာ ဖြတ်သန်းပြသလာခဲ့ကြသည်ကို မြင်တွေ့လိုက်ရသည်...

"ဒီတစ်ကြိမ်မှာတော့ ငါ တကယ်ပဲ မြင်တွေ့နိုင်လိမ့်မလား..?" သူ တိုးညှင်းစွာ ရေရွတ်လိုက်တော့သည်။
